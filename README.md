## Movie Magic

Simple Movie Searching App Created In React. Uses a Movie API to Display Searched Movie.

### Features
- Top Search Bar to Easily Search a Movie.
- Efficient Searching Pattern.
- Card Display of Movies.
- Responsive Design.
